# Candidate Assessment User Guide

## Introduction
The Candidate Assessment application is a secure, timed, proctored web platform designed for candidates to take their General Aptitude and Departmental tests. It ensures a fair testing environment through device-locking and strict anti-cheating mechanisms.

## How to Use

### 1. Launching the App
You will receive a unique assessment link via email or HR.
- **Launch:** Simply click the link to open the test in your web browser. 
- *Note: Ensure you open the link on the device (laptop/desktop/mobile) and browser you intend to use for the entire duration of the test.*

### 2. The Workflow (Step-by-Step)
1. **Verify Identity**: Wait for the "Verifying your link..." message to disappear. Once verified, you will see "Welcome, [Your Name]".
2. **Start Assessment**: Click the "Start Assessment" button to begin.
   - *Note: A strict anti-cheating mechanism will activate. DO NOT close or switch the browser tab.*
   - *Note: The countdown timer starts immediately once the test begins.*
3. **Navigate Sections**: Click the section buttons at the top of the screen to switch between "General Aptitude" and "Departmental Section" (if applicable to your test).
4. **Answer Questions**: 
   - For Multiple Choice: Click the radio button next to your chosen option.
   - For Descriptive: Type your detailed answer into the provided text box.
5. **Submit Assessment**: Click the "Submit Assessment" button at the bottom of the page when you are finished. The test will automatically submit if the timer reaches zero.

## Interface Reference

| Control / Input | Description | Expected Format |
| :--- | :--- | :--- |
| **Timer Display** | Shows the remaining time. Turns red during the last 5 minutes and flashes during the last 60 seconds. | MM:SS |
| **Start Assessment Button** | Begins the test, starts the timer, and activates proctoring. | Click / Tap |
| **Section Navigation** | Buttons to switch between "General Aptitude" and "Departmental Section". | Click / Tap |
| **Multiple Choice Options** | Radio buttons to select the correct answer for objective questions. | Single Selection |
| **Text Area** | Input field for typing out answers to descriptive questions. | Text String |
| **Submit Assessment Button** | Opens a confirmation dialog to finalize and submit your test. | Click / Tap |

## Troubleshooting & Validations

If you see an error, check this table:

| Message | What it means | Solution |
| :--- | :--- | :--- |
| "Invalid or Expired Token" / "This test link has expired." | Your test link is older than 24 hours or has been invalidated by the system. | Please contact HR for a new test link. |
| "This test is active on another device. Access denied." | The proctoring system locks the test to the original device and browser it was launched on. | Please return to the original device and browser to continue your test. |
| "This test has already been completed and cannot be reopened." | You have already submitted the test, or the timer expired on a previous attempt. | No action needed. Your results have already been recorded. |
| "⚠️ TAB SWITCH DETECTED" | The system detected that you moved away from the test tab (e.g., opened a new tab, minimized the browser, or checked notifications). | Return to the test immediately. You are allowed a maximum of 3 warnings before potential disqualification. |
| Screen turns black with "⚠️ SECURITY MODE ACTIVE" | The test window is in the background or inactive to prevent screenshotting. | Click or tap back into the test window to resume the assessment. |
